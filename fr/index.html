T-Rex Dino Coureur

Jeu en ligne depuis le navigateur Google Chrome

Le jeu avec un dinosaure sautant par-dessus des cactus et des ptérodactyles est apparu pour la première fois dans le navigateur Google Chrome Canary en 2014.
La page avec le jeu s'est ouverte lorsque votre connexion Internet a été interrompue. Au départ, le dinosaure ne fait rien sur la page, mais si vous appuyez sur la barre d'espace, la flèche vers le haut ou appuyez sur l'écran de votre téléphone, le jeu commencera.
Pendant longtemps à cause de cela, peu de gens connaissaient le jeu.

Cliquez sur l'écran (pour les appareils mobiles), sur "l'espace" ou la flèche vers le haut ou vers le bas pour lancer le jeu. Si le jeu ne fonctionne pas, essayez de l'ouvrir dans un navigateur alimenté par Chromium (Google Chrome, Microsoft Edge, Opera, Vivaldi, Brave).

À propos du jeu

Pour sauter comme un dinosaure, appuyez sur la barre d'espace ou la flèche vers le haut, appuyez sur l'écran si vous avez un appareil mobile.
Après le début du jeu, le dinosaure commencera à courir. Pour sauter par-dessus les cactus, vous devez appuyer à nouveau sur "espace / flèche vers le haut / toucher l'écran".
La vitesse du jeu augmentera progressivement et il deviendra de plus en plus difficile de sauter par-dessus les cactus. Lorsque vous marquez 400 points, des dinosaures volants - des ptérodactyles - apparaissent dans le jeu. Ils peuvent également être sautés ou, si vous jouez sur l'ordinateur, vous pouvez vous baisser en appuyant sur la flèche vers le bas.
Le jeu avec le genre «Runner» est sans fin, plus vous parvenez à tenir longtemps, plus c'est cool.

Le nom du dinosaure est T-Rex, qui est le nom d'une seule espèce de tyrannosaure, Tyrannosaurus rex. Rex signifie roi en latin. L'individu moyen était beaucoup plus grand que les ptérodactyles et les cactus.

Contrairement à d'autres services similaires, ici les "entrailles du jeu" sont fermées à l'accès gratuit. Par conséquent, les simples astuces du réseau ne fonctionneront pas. Cela sera particulièrement utile lorsque le système de classement entrera en vigueur.
